CampusIQ is a next-generation Smart Campus Operations Platform designed to unify academic, administrative, operational, and student services into a single intelligent ecosystem.

🚀 Getting Started
Clone Repository
git clone https://github.com/kiyotakaaKira/CAMPUS-IQ.git
cd CAMPUS-IQ
Install Frontend
npm install
Install Backend
cd backend
npm install
Configure Environment Variables

Create:

backend/.env

Example:

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

PORT=3001

OLLAMA_URL=http://localhost:11434
Start Backend
cd backend

npm start

Backend runs on:

http://localhost:3001
Start Frontend
npm run dev

Frontend runs on:

http://localhost:5173
