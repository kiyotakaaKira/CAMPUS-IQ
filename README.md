# CampusIQ

CampusIQ is a generalized Smart Campus Monitoring & Operations Intelligence platform. It shifts the focus from siloed dropout prediction to a comprehensive, institution-agnostic campus operating system.

## Overview
Built with a sleek, premium design philosophy (using React, Tailwind CSS, shadcn/ui, and Framer Motion), CampusIQ centralizes operational data across 14 modules including Student Hub, Faculty Workspace, Maintenance Operations, Transport Management, and more.

## Tech Stack
- Frontend: Vite + React 18 + TypeScript + Tailwind CSS
- Data Fetching: TanStack Query (React Query)
- Motion: Framer Motion
- UI Components: shadcn/ui (Radix)
- Routing: React Router v6

## Running Locally
```bash
npm install
npm run dev
```
